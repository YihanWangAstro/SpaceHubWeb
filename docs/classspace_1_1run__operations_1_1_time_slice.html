Create a wrapper on callable object(function pointer, functor, lambda) of which accepts one parameter. The type of the parameter must have method time(). The wrapped callable object will be invoked for equal spacing time interval for time provided by time(). This class is basically used to implement equal time operation in simulations. For example, one can provide a printer [](auto&p){std::cout << p << std::endl;} as the pre_step_operation in RunArgs to output the state of the integrated system. This printer then will be invoked before every step integration. The output might be very dense sometimes, thus outputting the result of every step is somewhat heavy. If one wraps the printer with TimeSlice, ‘TimeSlice([](auto&p){std::cout << p << ’
'}, 0, 100, 10)`, then the output will be performed only at p.time()=[0,100/10, 2*100/10, 3*100/10,...100].
